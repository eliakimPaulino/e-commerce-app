# Flutter Firebase eCommerce App

A professional eCommerce mobile application built with Flutter and powered by Firebase.

This project demonstrates modern mobile development practices, including authentication, cloud integration, local persistence, responsive UI, and clean application architecture.

Access the project here: https://flutter-ecommerce-app-eli-2026.web.app/

---

# Features

## Authentication
- Email & password authentication
- User registration and login
- Email verification
- Persistent user sessions

## Firebase Integration
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Real-time data synchronization

## User Profile
- View and update user profile
- Store user data securely in Firestore

## Shopping Features
- Product listing
- Cart management
- Wishlist functionality
- Offline local storage support

## UI/UX
- Modern and responsive interface
- Light & Dark mode
- Smooth user experience
- Mobile-first design

# Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/flutter-firebase-ecommerce-app.git

cd flutter-firebase-ecommerce-app
```

---

## 2. Install Dependencies

Make sure Flutter is installed on your machine.

```bash
flutter pub get
```

---

# Firebase Setup

## Create a Firebase Project

Go to:

https://console.firebase.google.com

---

## Enable Firebase Services

Enable:
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

---

## Android Configuration

Download the `google-services.json` file and place it inside:

```text
android/app/
```

---

# Run the Application

```bash
flutter run
```
